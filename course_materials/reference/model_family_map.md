# Model Family Map

This page links problems to model families in simple terms.

| Model Family | Plain Description | Typical Use |
|--------------|-------------------|-------------|
| Regression | Predicts a number | Sales, scores, prices |
| Classification | Assigns a label | Spam, topic, risk level |
| Clustering | Groups similar cases | Customer groups, behavior patterns |
| Sequence Model | Handles ordered data | Time series, language, events |
| Recommendation Model | Suggests items | Products, content, resources |
| Anomaly Model | Flags unusual cases | Fraud, faults, outliers |
