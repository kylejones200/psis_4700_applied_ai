# Problem to Model Examples

Use this table to match problems to model families.

| Problem | Best Model Family | Why It Fits |
|---------|------------------|-------------|
| Predict next month sales | Regression | Output is a number |
| Decide if an email is spam | Classification | Output is a label |
| Group customers by behavior | Clustering | Output is natural groupings |
| Predict equipment failure time | Sequence Model | Data unfolds across time |
| Recommend articles for readers | Recommendation Model | Output suggests items |
| Detect unusual credit card activity | Anomaly Model | Output highlights rare events |
