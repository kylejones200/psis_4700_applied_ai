\section*{Purpose}

This guide helps you decide when AI adds value and when human work matters more.

\vspace{15pt}

\section*{\textcolor{yescolor}{Use AI When}}

\begin{yesbox}
\begin{itemize}[leftmargin=20pt]
    \item The task needs drafts or options
    \item The task needs structure or summaries
    \item The task supports exploration or early thinking
    \item The task has low risk and clear review
\end{itemize}
\end{yesbox}

\section*{\textcolor{nocolor}{Do Not Use AI When}}

\begin{nobox}
\begin{itemize}[leftmargin=20pt]
    \item The task requires expert judgment
    \item The task affects safety or legal outcomes
    \item The task involves sensitive or private data
    \item The task demands verified facts without error
\end{itemize}
\end{nobox}
