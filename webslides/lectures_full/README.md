# Full Semester Lectures

Formal lecture presentations for a complete semester-long course format.

## Lecture Series

1. **lecture1_What-Is-Applied-AI.md** - Course framing, AI definitions, and expectations
2. **lecture2_Data-and-Models.md** - Data fundamentals and model basics
3. **lecture3_Working-with-Text.md** - Natural language processing and text analysis
4. **lecture4_Vision-and-Images.md** - Computer vision and image processing
5. **lecture5_AI-in-Business.md** - Business applications and value creation

## Format

These lectures are designed as full 50-75 minute presentations with:
- Detailed slide content
- Speaker notes
- Visual examples
- Exercises and activities

## Usage

Use these for:
- Traditional semester courses (15-16 weeks)
- Guest lectures in other courses
- Professional development workshops
- Faculty training sessions

## Relation to Other Materials

These lectures cover similar content to the 8-week intro course but in a more traditional academic lecture format with deeper coverage per session.
