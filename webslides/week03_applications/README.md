# Week 3: Applications Across Domains

## Overview
Explore AI applications across sectors via short case studies. Evaluate feasibility, risks, and value. Prepare a brief domain analysis.

## Learning Objectives
- Identify common AI application patterns in multiple domains
- Evaluate feasibility using value/cost/risk and data readiness
- Draft a short case analysis using a provided template
